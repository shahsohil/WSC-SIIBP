Weakly Supervised, Constrained & Stacked Integrative Indian Buffet Processs (WSC-SIIBP) CODE
=============================================================================================

README for Matlab code of "Weakly Supervised Learning of Heterogeneous Concepts in Videos", ECCV 2016.

This code is adopted from the code of "Weakly Supervised Learning of Objects, Attributes and their Associations", ECCV 2014.
==========================================================================

DESCRIPTION
==========================================================================
This code is the final implementation used for reporting results in the paper.
It includes code for WS-SIIBP, WSC-SIIBP and WSC-SIBP
(Please refer paper for description of each)
